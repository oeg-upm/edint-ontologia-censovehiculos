Introducción volver a índice

Esta ontología forma parte del catálogo de ontologías que está siendo desarrollado en el contexto del Espacio de Datos para las Infraestructuras Urbanas Inteligentes (EDINT). Asimismo, fue identificada como una ontología relevante para la representación de datos gestionados por entidades locales en la guía Datos abiertos FEMP 2019: 40 conjuntos de datos a publicar por las Entidades Locales, tal y como se describe también en la correspondiente nota de datos.gob.es, y por tanto recogido en el repositorio Github de apoyo a la iniciativa, que queda deprecado una vez se lanza este repositorio.
