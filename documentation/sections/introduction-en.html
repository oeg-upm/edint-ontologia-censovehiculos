Introduction back to ToC

This ontology is part of the ontology catalogue that is being developed in the context of the Data Space for Smart Urban Infratructures (EDINT). Moreover, it was identified as a relevant ontology to represent data management by local entities in the FEMP open data guide Datos abiertos FEMP 2019: 40 conjuntos de datos a publicar por las Entidades Locales, as described also at this datos.gob.es press release, and hence covered by the Github repository to support the initiative. That repository will be deprecated.
